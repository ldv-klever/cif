CIF
===

:abbr:`CIF (C Instrumentation Framework)` is a tool that implements
`aspect-oriented programming <http://en.wikipedia.org/wiki/Aspect-oriented_programming>`__ for the C programming
language.
You can learn more about CIF at the `project site <https://forge.ispras.ru/projects/cif>`__.
